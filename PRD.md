# PRD · 彤彤生日网站

> **所有代码生成必须遵循此文档风格规范。每次生成前先阅读。**

---

## 一、项目概述

为爱人彤彤制作的中式风格生日纪念网站。移动端优先，单页滚动叙事，7 个章节递进展开。

## 二、核心风格

### 设计基调
- **中式留白**：大面积留白，内容居中，不堆砌
- **温润含蓄**：暖色调宣纸背景，朱红点缀，烫金提亮
- **克制不花哨**：每屏只一个视觉重点，装饰极弱化
- **移动优先**：max-w-md 居中，所有交互适配触屏

### 绝对禁止
- ❌ 五颜六色的背景或装饰
- ❌ 彩色渐变滥用（除烫金文字效果外）
- ❌ 自绘具象 SVG（蛋糕、花卉等）做主视觉
- ❌ 高饱和度色块大面积铺底
- ❌ 任何"山寨感"的装饰元素

## 三、色彩规范

| 名称 | 色值 | 用途 |
|------|------|------|
| 宣纸底 | `#F5EDE0` | 主背景色 |
| 暖宣纸 | `#FFF7EA` → `#EFE3D0` | 封面/卷轴渐变 |
| 墨色 | `#1F1B1A` | 正文文字 |
| 朱红 | `#B03A48` | 印章、强调色、分隔线 |
| 烫金 | `#C9A368` → `#F1E0B0` | 标题烫金效果 |
| 夜色 | `#0F1B2A` → `#080E18` | 终章背景 |
| 雾灰 | `#8B8278` | 次要文字 |

**原则**：整站只有 **宣纸色 + 墨色 + 朱红 + 烫金** 四色体系，夜色章节额外用深蓝黑。不允许引入其他色相。

## 四、字体规范

| 名称 | CSS 变量 | 用途 |
|------|----------|------|
| 霞鹜文楷 | `--font-kai` | 标题、诗句、信件 |
| 思源宋体 | `--font-song` | 正文、说明文字 |
| 英文衬线 | `--font-en` | 章节英文标签 |

## 五、章节结构

1. **封面 Cover** — 合照 + 大字"彤彤" + 长按进入
2. **序章 Prelude** — 卷轴展开 + 诗句
3. **初见 Meet** — 早期合照 + "山有木兮"
4. **永结 Wedding** — 婚纱照 + 婚誓
5. **初啼 Baby** — 宝宝降生 2025.4.15
6. **日常 Together** — 旅行/生活照 + 诗句
7. **生日 Birthday** — 大字"彤" + 照片 + 一封信
8. **终章 Finale** — 夜色 + 蜡烛 + 许愿 + 印章落款

## 六、装饰规范

### 允许的装饰
- 纯 CSS 蜡烛（半透明烛体 + 多层光晕火焰）
- SVG 印章（朱红底 + noise filter 模拟朱泥磨损）
- 水墨分隔线（极低 opacity ≤ 0.25）
- 四角花卉（opacity ≤ 0.15，仅封面用）
- 卷轴杆（细线 1.5px，低饱和金色）

### 装饰原则
- 所有装饰 opacity 不超过 0.25（印章除外）
- 装饰是背景，不能抢内容
- 宁可没有，不要山寨
- 如果需要高质量装饰素材，使用真实 PNG 图片，不用自绘 SVG

## 七、文案规范

- **不写真实年龄**，用"永远十八"、"芳华"等表达
- 语气温柔、私密、第一人称
- 中文为主，英文仅做章节标签点缀
- 信件用霞鹜文楷，行距宽松
- 所有文案集中在 `lib/manifest.ts` 管理

## 八、交互规范

- 长按门扉进入（EntryGate）
- 背景音乐开关（BgmToggle）
- 花瓣飘落（Petals canvas）
- 卷轴滚动展开（Framer Motion useScroll）
- 蜡烛点击/吹气熄灭（Candles）
- 烟花 canvas（Finale）
- 所有滚动动画用 Reveal 组件统一处理

## 九、技术栈

- Next.js 14 App Router + TypeScript
- TailwindCSS（自定义色彩/字体）
- Framer Motion（动画/滚动）
- Lenis（平滑滚动）
- 纯 Canvas（花瓣/烟花）

## 十、素材原则

- 照片由用户自行放入 `public/media/photos/`
- 视频由用户自行放入 `public/media/videos/`
- 音乐由用户自行放入 `public/media/audio/`
- 缺失素材时显示优雅占位框，不报错
- **不随意从网上下载图片充当背景**，CSS 纯色/渐变比不合适的图片好
